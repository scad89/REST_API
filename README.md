# REST_API

Mini project for learning how to work with DRF.

Technology stack:

```
- Django
- Django Rest Framework
- PostgreSQL
- Psycopg2
- Djoser
- Simple JWT
- Docker
- Docker-Compose
- Gunicorn
- dotenv
- drf yasg
- pip compile multi
```

## Usage

Run services with env file:
`docker-compose up --build`
